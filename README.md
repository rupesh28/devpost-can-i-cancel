# My Side Panel Extension

This is a boilerplate for a Chrome extension that uses the side panel API.

## How to load the extension in Chrome

1.  Open Chrome and navigate to `chrome://extensions`.
2.  Enable "Developer mode" by toggling the switch in the top right corner.
3.  Click on the "Load unpacked" button.
4.  Select the directory where you have saved the extension files.
5.  The extension should now be loaded and you should see its icon in the Chrome toolbar.
6.  Click on the extension icon to open the side panel.
